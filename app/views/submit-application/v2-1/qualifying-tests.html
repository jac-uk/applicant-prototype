Qualifying tests

We use online qualifying tests to create a shortlist of candidates to go through to the next stage of the selection process.

There are 3 types of qualifying tests.

  1. Situational judgement test.
  2. Critical analysis test
  3. Scenario test.

We’ll email you some time before the test date to tell you which ones you’ll need to take and when you’ll need to take them.

1. Situational judgement test

You’ll have a set time to answer a number of multiple-choice questions. Each question will present a situation you might face in the role you’ve applied for.

There are 5 possible answers for each question. You’ll need to choose which you think are the best and worst answer in each case. There are no marks for next best or next worst answer.

Tests are scored automatically. You’ll need to answer at least 30% of questions correctly.

2. Critical analysis test

We’ll send you a legal judgment or text before the test date. During the test, you’ll have a set time to answer a number of multiple-choice questions about this text. Each questions will have a choice of 5 answers. You’ll have to decide which is the most accurate.

Tests are scored automatically. You’ll need to answer at least 30% of questions correctly.

3. Scenario test

You’ll have a set time to answer questions about previous decisions on different appeal cases. You’ll be given the decision texts and grounds of appeal during the test.

Word limits will vary by question, but each will be clearly marked. There are also different maximum scores for each question. Again, these will be clearly marked.

Answers are marked by JAC panellists. These are validated to make sure marking is consistent across all panellists.

Test dates

We try to avoid holding qualifying tests during school and judicial holiday dates. It’s not possible to avoid all religious festivals.

If exceptional circumstances mean you cannot take a test on the specified date, we’ll arrange a second test date.
